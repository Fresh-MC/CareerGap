# NoGap Dashboard

The NoGap Dashboard is a graphical user interface (GUI) application designed to interact with the NoGap ecosystem. It provides a user-friendly way to manage and utilize the functionalities of the NoGap core library and CLI.

## Features

- Intuitive interface for managing NoGap functionalities.
- Integration with the NoGap CLI for command execution.
- Playwright-based end-to-end testing for ensuring application reliability.

## Getting Started

To get started with the NoGap Dashboard, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd nogap-workspace/nogap_dashboard
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the application**:
   ```
   npm start
   ```

## Development

For development purposes, you can run the application in development mode, which provides hot reloading and other features.

## Testing

Playwright tests are set up to ensure the GUI functions as expected. To run the tests, use the following command:

```
npm run test
```
